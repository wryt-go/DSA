## Problem Format

Each problem is organized in the following format:

1. Problem Statement
2. Input/Output Format
3. Sample Test Cases
4. Starter Code with Comments
5. Hints/Explanation

## Topics Covered

- Arrays
- Linked Lists
- Stacks
- Queues
- Recursion
- Backtracking
- Hashing
- Searching
- Sorting

## Difficulty Levels

- Easy: Problems suitable for beginners
- Medium: Problems that require more complex thinking and implementation

## How to Use

1. Navigate to the topic you want to practice
2. Choose a difficulty level
3. Read the problem statement and requirements
4. Try to solve the problem using the provided starter code
5. Check your solution against the sample test cases
